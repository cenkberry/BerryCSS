BerryCss enables you to construct your project more efficiently by merging the structures of Bootstrap and Tailwind.
At the same time, BerryCss includes a global CSS reset feature, enhancing the efficiency of your project construction.
Documentation
Available breakpoints
BerryCSS includes six default breakpoints, sometimes referred to as grid tiers, for building responsively. These breakpoints can be customized if you’re using our source Sass files.
// `$xxs` applies to small devices (Landscape phones, more than 374px)
$xxs: 374px;
@media screen and (min-width: $xxs) {...}
// `$mob` applies to general phone devices (phones, more than 576px)
$mob: 576px;
@media screen and (min-width: $mob) {...}
// `$tab` applies to general tablet devices (tablets, more than 767px)
$tab: 767px;
@media screen and (min-width: $tab) {...}
// `$lap` applies to general laptop devices (laptops, more than 1023px)
$lap: 1023px;
@media screen and (min-width: $lap) {...}
// `$lap` applies to general big screen devices (big screens, more than 1440px)
$desk: 1440px;
@media screen and (min-width: $desk) {...}
// `$lap` applies to general large screen devices (large screens, more than 1440px)
$big: 1600px;
@media screen and (min-width: $big) {...}

Default Container
Our default .container class is a responsive, fixed-width container, meaning its max-width changes at each breakpoint.

Responsive Size System
We've created a responsive 'size system' and applied it to all classes. Without any extra effort, all pixel values will automatically scale down to a ratio of 0.81 below the tablet breakpoint of 767px, achieving a fully responsive appearance automatically.
The provided values for font size, margin, padding, gap, radius, etc., will scale down by a factor of 0.81, and there will be no need to write individual media queries.

Flexbox Classes
There's no need to define CSS values for flexbox properties separately. We've encapsulated them into classes for easy use.
For example, when you type 'df', it automatically sets "display:flex;", and similarly, typing 'aic' would set "align-items:center;".You can individually verify the examples below.
// Align Items Classes
ais = align items start
aic = align items center
aie = align items end
// Justify Content Classes
jcs = justify content start
jcc = justify content center
jce = justify content end
// Align & Justify Classes
class="df aic jcc"
class="df aic jce"
// Center Class
class="center"
